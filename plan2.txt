PROJECT TITLE:
SafeSteer-IN
Inference-Time Safety Steering for Indic LLMs using Contrastive Activation Addition (CAA)

OBJECTIVE:
Build a fully functional prototype demonstrating inference-time safety steering
on Indic Large Language Models using open-source tools and Azure services.

The system must:
1. Detect harmful prompts in Indic languages.
2. Apply steering vectors dynamically at inference time.
3. Show baseline vs steered outputs.
4. Integrate Azure AI Content Safety for independent scoring.
5. Track experiments via Azure ML.
6. Provide a Gradio demo UI.
7. Be fully reproducible.

SECTION 5: STEERING VECTOR EXTRACTION
------------------------------------------------------------

Use TransformerLens to capture residual stream activations.

Target Layers:
Layer 12 (primary)
Layer 14 (secondary)

Procedure:

For each (language, category):

1. Load prompts.
2. Tokenize safe prompts.
3. Run model forward pass with cache.
4. Extract residual activations:
   cache["blocks.{layer}.hook_resid_pre"]
5. Average activations across safe prompts.
6. Repeat for unsafe prompts.
7. Compute steering vector:

   steering_vec = mean_unsafe - mean_safe

8. Normalize:

   steering_vec /= torch.norm(steering_vec)

9. Save:

steering_vectors/{language}_{category}_layer{layer}.pt

Use Azure ML compute cluster for parallel extraction jobs.

Track:
- category
- layer
- vector norm
- dataset version

SECTION 7: INFERENCE-TIME STEERING ENGINE
------------------------------------------------------------

Mechanism:
Use forward hooks.

Function:

def steering_hook(resid, hook):
    resid += alpha * steering_vector
    return resid

Attach hook to:
model.blocks[layer].hook_resid_pre

Workflow:

1. User submits prompt.
2. Run risk classifier.
3. If risk_score > threshold:
     load corresponding steering vector.
     attach hook.
4. Generate response.
5. Remove hook.

Alpha Calibration:

Test values:
[0.5, 1.0, 2.0, 4.0]

Select best tradeoff:
maximize safety reduction
minimize perplexity increase
